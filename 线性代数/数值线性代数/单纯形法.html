单纯形法

问题概述

待求解的问题具有如下的一般形式：

min⁡x∈RncTxs.t. Ax=b,      x≥0.\begin{aligned} &\mathop{\min}\limits_{x \in \mathbb{R}^n} c^T x\\ &\text{s.t. } Ax = b, \\ &\quad \;\;\;x \geq 0. \end{aligned} ​x∈Rnmin​cTxs.t. Ax=b,x≥0.​

单纯形算法

给定任意基本指标集合 B⊂[n]\mathcal{B} \subset [n]B⊂[n] 与非基本指标集合 N=[n]−B\mathcal{N} = [n] - \mathcal{B}N=[n]−B.

Repeat

矩阵 BBB 由系数矩阵 AAA 中的部分列向量组成，这些列向量的下标即是指标集 B\mathcal{B}B，矩阵 NNN 同理. 向量 cBc_BcB​ 是由 ccc 中的部分元素组成的列向量，cNc_NcN​ 同理.

计算 xB=B−1b,  xN=0⃗x_B = B^{-1}b,\; x_N = \vec{0}xB​=B−1b,xN​=0.

求解 BTλ=cBB^T \lambda = c_BBTλ=cB​，计算 sN=cN−NTλs_N=c_N - N^T \lambdasN​=cN​−NTλ.

if sN≥0⃗s_N \geq \vec{0}sN​≥0

   stop;(找到了最优解为 xB+xNx_B + x_NxB​+xN​ )

else

   Solve Bd=AqBd = A_qBd=Aq​

     stop;(问题无解)

   else
